# English Book Reader App

## 개요

이 애플리케이션은 사용자가 PDF 형식의 영어 책(또는 다른 언어의 책)을 더 쉽게 읽을 수 있도록 돕습니다. 텍스트를 한 번에 한 문장씩 보여주고, 문맥에 맞는 단어 정의를 제공하며, 어휘력 향상 및 이해도 퀴즈 기능을 포함합니다.

## 기능

### 핵심 읽기 경험
- **PDF 업로드:** 사용자는 PDF 파일을 업로드할 수 있습니다.
- **문장 단위 표시:**
    - 한 번에 하나의 문장이 강조 표시됩니다.
    - 나머지 텍스트는 흐리게 처리됩니다.
    - 사용자는 문장 사이를 탐색할 수 있습니다.

### 어휘 및 학습
- **AI 기반 문맥 내 단어 정의:** 단어를 1.5초간 길게 누르면 Gemini AI가 문맥에 맞는 단어의 뜻을 한국어로 제공합니다.
- **단어장:** 모르는 단어를 개인 단어장에 저장할 수 있습니다. (Firebase 연동 예정)
- **AI 기반 퀴즈:**
    - 저장된 단어를 기반으로 한 단어 퀴즈.
    - 읽은 내용에 대한 이해도를 확인하는 AI 생성 퀴즈.

### 클라우드 동기화 (Firebase)
- **사용자 인증:** Google 계정을 통한 로그인.
- **데이터 지속성:** 단어장 및 읽기 진행 상황을 Firestore에 저장하여 여러 기기에서 동기화.

## 구현된 기능

*   **프로젝트 설정:** 기본 HTML, CSS, JavaScript 파일 생성.
*   **PDF 파싱 및 문장 표시:** PDF.js를 이용한 텍스트 추출 및 문장 단위 UI/UX.
*   **단어 상호작용 및 단어장:** 단어 선택, 정의 모달, 로컬 단어장 추가 기능.
*   **AI 기반 기능:**
    *   Gemini API를 연동하여 문맥에 맞는 단어 뜻 제공.
    - AI가 생성하는 내용 이해 퀴즈 기능 구현.

## 현재 계획

1.  **Firebase 통합 (진행 중):**
    -   `index.html`에 Firebase SDK(Auth, Firestore) 스크립트를 추가합니다.
    -   사용자가 Firebase 구성 객체를 붙여넣을 수 있는 UI를 추가합니다.
    -   인증 및 데이터베이스 상호작용을 관리할 `firebase.js` 모듈을 생성합니다.
    -   Google 계정으로 로그인/로그아웃하는 기능을 구현합니다.
    -   로그인 시 사용자의 단어장을 Firestore에서 불러오고, 단어 추가 시 Firestore에 저장하도록 로직을 수정합니다.
    -   사용자의 읽기 진행률(현재 문장 인덱스)을 Firestore에 저장하고 동기화합니다.
2.  **전반적인 디자인 개선:** 앱을 더욱 편리하고 매력적으로 보이도록 다듬습니다.
